### Maven 依赖

Maven依赖GAV:

```xml

<dependency>
    <groupId>com.uniubi.cloud.athena</groupId>
    <artifactId>java-sdk-parent</artifactId>
    <version>${latestTagName}</version>
</dependency>
```

请使用最新的RELEASE tag名称作为版本号
