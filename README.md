### Maven dependency
Maven dependency configuration:
```xml

<dependency>
    <groupId>com.uniubi.cloud.athena</groupId>
    <artifactId>java-sdk-parent</artifactId>
    <version>${latestTagName}</version>
</dependency>
```
Use the latest release tag name as the version.
